# 105. Construct Binary Tree from Preorder and Inorder Traversal

Given two integer arrays preorder and inorder where preorder is the preorder traversal of a binary tree and inorder is the inorder traversal of the same tree, construct and return the binary tree.

 

> **Example 1**
> 
> <img src="../../imgs/105-1.jpg" alt="105" width="300"/>
> 
> Input: preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
> 
> Output: [3,9,20,null,null,15,7]

> **Example 2**
> 
> Input: preorder = [-1], inorder = [-1]
> 
> Output: [-1]


---

위 문제는 우선 Inorder와 Preorder에 대한 이해가 필요하다.

사고의 흐름은 아래와 같다.

1. 각각의 구조를 통해 알 수 있는 것은? 
1) Preorder : root
2) Inorder : 구조 나눔 

둘의 형태를 이해했다면, **Subtree**를 계속 쪼개야 하는 형태임을 알 수 있다.

즉, Subtree 이므로 DFS 문제로 접근 할 수 있음을 알 수 있는 것. 
(BFS로 진행한다면 순서 유지가 불가능하므로 이용하기 불편함)

2. Subtree는 어떤 식으로? 
1) root 생성
2) left tree 생성
3) right tree 생성

```cpp
TreeNode * root = new TreeNode();
root->left = dfs();
root->right = dfs();
```

3. 어디를 기준으로 Subtree를 분리해야 할까?
-> Preorder로 얻은 root->value 를 이용하여, inorder에서 해당 위치를 찾는다.
이후 찾아낸 index의 좌/우 영역이 각각 Left/Right subtree가 된다. 


---

### Solution 
```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
unordered_map<int,int> um;
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        
        int preorder_index = 0;
        for(int i=0; i<inorder.size(); i++){
            um[inorder[i]]=i;
        }
        return dfs(preorder, preorder_index, 0, preorder.size()-1);
    }
    TreeNode* dfs(vector<int>& preorder, int& preorder_index, int left_index, int right_index){
        if(left_index > right_index) return nullptr;
        TreeNode* root = new TreeNode(preorder[preorder_index]);
        preorder_index ++;
        int mididx = um[root->val];
        root->left = dfs(preorder, preorder_index, left_index, mididx-1);
        root->right = dfs(preorder, preorder_index, mididx+1, right_index);
        return root;
    }
};
```

4. 핵심 코드 
```cpp
TreeNode* root = new TreeNode(preorder[preorder_index]);
```
여기서 preorder_index를 쓰는 이유는, preorder이므로 index순서 그대로 Stubtree의 root이 되기 때문임.

```cpp
int mididx = um[root->val];
```
속도를 빠르게 하기 위한 hashmap.
만약 vector를 사용한다고 하면, 선형탐색이 되므로 총 Time complexitiy 가 O(N^2)이 되어버린다. 
(hasmap 이용시 O(N))
