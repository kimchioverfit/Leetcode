# 560. Subarray Sum Equals K

Given an array of integers nums and an integer k, return the total number of subarrays whose sum equals to k.

A subarray is a contiguous non-empty sequence of elements within an array.

 

> **Example 1**
> 
> Input: nums = [1,1,1], k = 2
> Output: 2


> **Example 2**
> 
> Input: nums = [1,2,3], k = 3
> Output: 2

---

우선 **Subarray** 란?
-> 연속적인 array 형태. 즉, 특정 원소를 하나하나 보는 것이 아니라, 구간통쨰로 보면 된다. 

1. 연속적인 구간의 합은 어떤 식을 이용할까?
1) Bruteforce -> O(N^2)
2) 구간의 합이므로 Prefixsum -> Sum = prefix[right] - prefix[left-1] 형태로 접근이 가능함. 

2. 빠르게 접근 하기 위해서는 ?
Target number 인 K 를 바로 만족하지는 않더라도, 
Prefixsum - K 가 과거에 등장했다면 가능하다는 것을 인지해야함. 

즉, Two Sum 문제를 Prefixsum 과 합친 문제라고 보면 된다. 
unordered_map 이용을 하는 것이 관건.

### Solution 
```cpp
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        unordered_map<int,int> um;
        int prefixsum = 0;
        int cnt = 0;
        for(int i=0; i<nums.size(); i++){
            prefixsum += nums[i];
            if(prefixsum == k) cnt++;
            if(um.find(prefixsum-k)!=um.end()){
                cnt += um[prefixsum-k];
            }
            um[prefixsum] ++;
        }
        return cnt;
    }
};
```
