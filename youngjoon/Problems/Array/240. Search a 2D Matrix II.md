# 240. Search a 2D Matrix II

Write an efficient algorithm that searches for a value target in an m x n integer matrix matrix. This matrix has the following properties:

Integers in each row are sorted in ascending from left to right.
Integers in each column are sorted in ascending from top to bottom.

---

선행 문제인 74. Search a 2D Matrix 의 upgrade version

1. 어떤 구조로 접근해야할까?

기존과 달리, 전체 정렬이 아닌 일부 정렬상태 (Partially ordered Matrix).

이 경우, 전체 대상으로 Binary search 를 쓸 수 없다!

-> 각 row/col 에 대해서 Binary search 를 쓰자.
(하지만, O(m log n)으로, Optimal 은 아님)

2. 어떤 식으로 Binary search를 써야할까?
-> 특성을 파악하고 접근하자. 
matrix[x][y] > target 인 경우, 열 전체 사용 불가. y--;
matrix[x][y] < target 인 경우, 행 전체 사용 불가. x++;

---

### Solution 

```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();      // 행 개수
        int n = matrix[0].size();   // 열 개수
        
        int row = 0;                // 첫 번째 행부터 시작
        int col = n - 1;            // 마지막 열부터 시작
        
        while (row < m && col >= 0) {
            if (matrix[row][col] == target) {   // target을 찾은 경우
                return true;
            } else if (matrix[row][col] > target) { // target보다 큰 경우
                col--;      // 열을 왼쪽으로 이동
            } else { // target보다 작은 경우
                row++;      // 행을 아래로 이동
            }
        }
        return false;   // target이 없으면 false 반환
    }
};
```
