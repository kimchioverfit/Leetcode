# 207. Course Schedule

There are a total of numCourses courses you have to take, labeled from 0 to numCourses - 1. You are given an array prerequisites where prerequisites[i] = [ai, bi] indicates that you must take course bi first if you want to take course ai.

For example, the pair [0, 1], indicates that to take course 0 you have to first take course 1.
Return true if you can finish all courses. Otherwise, return false.

 

> **Example 1**
> 
> Input: numCourses = 2, prerequisites = [[1,0]]
> Output: true
> Explanation: There are a total of 2 courses to take. 
> To take course 1 you should have finished course 0. So it is possible.

> **Example 2**
> 
> Input: numCourses = 2, prerequisites = [[1,0],[0,1]]
> Output: false
> Explanation: There are a total of 2 courses to take. 
> To take course 1 you should have finished course 0, and to take course 0 you should also have finished course 1. So it is impossible.

---

### 1. 이러한 <u>선 후 관계</u>가 있는 경우에서, 우선 고려해야 할 것은?
-> 선후관계는 방향그래프를 의미하므로, **Cycle** 이 있는지를 판단해야 한다.

### 2. Cycle 은 어떻게 찾을까?
1) DFS 
2) BFS (Topological Sort, Kanh's Algorithm)
3) Union-Find (Data Structure) : 서로소 집합을 효율적으로 표현하기 위한 자료구조.

Union-Find 는 보통 무방향 그래프에서의 cycle detection에 적합하므로, 여기서는 탈락.



### 3.1. DFS를 이용한 접근
이러한 문제를 DFS 로 접근할 때 Visited vector를 보통 이용하는데, 
단순히 visited 만으로는 완전히 끝난 node인지 경로 상에 있는 node인지 판별할 수가 없다.
-> 현재 호출 Stack의 Mode 들을 기록하는 Vector가 따로 필요하다.

### 3.2. BFS를 이용한 접근 (정석적인 접근)

Kahn's Algorithm (Topological Sort)이 가장 적합하다. 

기본적으로 Cycle이 있으면 Sorting자체가 실패하므로 Cycle Detection 도 들어있음. 

순서는 아래와 같다

1. 전체 node 의 indegree를 체크. 
2. indegree가 0 인 지점을 찾아 Queue에 삽입. 
3. Queue 가 empty 될 때 까지 front를 빼내면서, 꺼낸 node에 연결된 edge를 제거한다.
4. 1~3을 반복한다.

### Solution (Kahn's Algorithm)

```cpp
class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        // 1. Count numbers of all node's indegree
        vector<int> nodeIndegree(numCourses, 0);
        vector<vector<int>> graph(numCourses); // To make adjacency list 

        for(int i = 0; i < prerequisites.size(); i++){
            int course = prerequisites[i][0];
            int pre = prerequisites[i][1];

            graph[pre].push_back(course); // pre -> course
            nodeIndegree[course]++;
        } 

        // insert 0 indegree Node into queue
        queue<int> q;
        check_zero_indegree(nodeIndegree, q);

        int count = 0;

        while(!q.empty()){
            int temp = q.front();
            q.pop();
            count++;

            for(int i = 0; i < graph[temp].size(); i++){
                int next = graph[temp][i];

                nodeIndegree[next]--;

                if(nodeIndegree[next] == 0){
                    q.push(next);
                }
            }
        }

        return count == numCourses;
    }

    void check_zero_indegree(vector<int>& nodeIndegree, queue<int>& q){
        for(int i = 0; i < nodeIndegree.size(); i++){
            if(nodeIndegree[i] == 0) q.push(i);
        }
    }
};
```
