## 학습 흐름

### 1. DFS / BFS

DFS와 BFS는 대부분의 그래프 문제의 기본 도구다.

- DFS: 깊게 들어가면서 탐색한다. 재귀, visited, recursion stack과 연결된다.
- BFS: 가까운 노드부터 탐색한다. queue, level, 최단 거리, grid 확산 문제와 연결된다.

먼저 풀기 좋은 문제:

- [200. Number of Islands](200.%20Number%20of%20Islands.md)
- [695. Max Area of Island](695.%20Max%20Area%20of%20Island.md)
- [994. Rotting Oranges](994.%20Rotting%20Oranges.md)
- [130. Surrounded Regions](130.%20Surrounded%20Regions.md)
- [133. Clone Graph](133.%20Clone%20Graph.md)

### 2. Connected Component / Tree

연결 요소는 DFS/BFS로 같은 그룹에 속한 노드들을 묶는 개념이다. Tree는 cycle이 없는 connected graph로 볼 수 있으므로 그래프의 특수 케이스로 이해하면 좋다.

관련 문제:

- [547. Number of Provinces](547.%20Number%20of%20Provinces.md)

### 3. Cycle / SCC / DAG / Topological Sort

방향 그래프에서는 단순 visited만으로 부족한 경우가 많다. 현재 탐색 경로에 있는 노드인지, 완전히 처리된 노드인지 구분해야 한다.

- Cycle: 방향/무방향 그래프에서 cycle 존재 여부를 판단한다.
- SCC: 방향 그래프에서 서로 도달 가능한 노드 그룹을 찾는다.
- DAG: cycle이 없는 방향 그래프다.
- Topological Sort: DAG에서 선후 관계를 만족하는 순서를 만든다.

SCC를 먼저 보면 DAG가 더 잘 이해된다. SCC들을 하나의 노드로 압축하면 전체 그래프는 DAG가 된다.

관련 문제:

- [207. Course Schedule](207.%20Course%20Schedule.md)
- [3481. Apply Substitutions](3481.%20Apply%20Substitutions.md)

### 4. Minimum Path

최단 경로는 그래프의 edge weight 조건에 따라 알고리즘을 선택한다.

- 가중치 없음: BFS
- 양수 가중치: Dijkstra
- 음수 가중치 가능: Bellman-Ford
- 모든 쌍 최단 경로: Floyd-Warshall

관련 문제:

- [909. Snakes and Ladders](909.%20Snakes%20and%20Ladders.md)
- [2359. Find Closest Node to Given Two Nodes](2359.%20Find%20Closest%20Node%20to%20Given%20Two%20Nodes.md)
- [3342. Find Minimum Time to Reach Last Room II](3342.%20Find%20Minimum%20Time%20to%20Reach%20Last%20Room%20II.md)

### 5. Disjoint Set / Union-Find / MST

Disjoint Set은 여러 노드가 같은 집합에 속하는지 빠르게 확인하는 자료구조다. Union-Find는 Disjoint Set을 구현하는 대표적인 방식이다.

MST의 Kruskal 알고리즘은 Union-Find를 사용하므로, MST보다 Union-Find를 먼저 보는 것이 좋다.

관련 문제:

- [305. Number of Islands II](305.%20Number%20of%20Islands%20II.md)
- [547. Number of Provinces](547.%20Number%20of%20Provinces.md)
